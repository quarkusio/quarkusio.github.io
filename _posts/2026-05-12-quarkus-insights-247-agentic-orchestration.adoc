== The Journey: From Spring Boot to Quarkus

Jonathan's journey began with a home lab project using Spring Boot and Spring AI. After attending a Connecticut Java Users Group talk by Eric Deandrea (from the Quarkus team), he was inspired to explore LangChain4j with Quarkus. Using https://claude.ai/[Claude AI] for code conversion, he successfully migrated his Spring Boot application to Quarkus with LangChain4j in approximately one hour.

Key migration benefits observed:
- Faster startup times
- Reduced memory footprint
- Lower CPU usage
- Native compilation with https://www.graalvm.org/[GraalVM] support

== Distributed Agentic Architecture: A Different Approach

Rather than using traditional LangChain4j annotations for sequential or parallel agent pipelines within a single application, Jonathan implemented a **distributed round-table architecture** using https://nats.io/[NATS] messaging:

=== Traditional Approach (Monolithic)
[source,java]
----
@SequenceAgent
@ParallelAgent
@SupervisorAgent
// All agents in one compiled application
----

=== Distributed Approach (Kubemoot)
- **Agents as Kubernetes pods**: Each agent runs as an independent container
- **NATS messaging**: Agents communicate via distributed messaging (not inter-process)
- **Democratic coordination**: Agents self-select to contribute based on their expertise
- **Round-table pattern**: Similar to Reddit threads where experts chime in

=== Architecture Components

1. **Coordinator Agent**: Interprets incoming questions and broadcasts to the agent pool
2. **Specialist Agents**: Each has domain expertise (Kubernetes, Proxmox, SQL, etc.)
3. **MCP Tools**: Agents have access to https://modelcontextprotocol.io/[Model Context Protocol] tools
4. **Prompts as ConfigMaps**: Stored as Kubernetes YAML manifests, not compiled code
5. **Kubemoot Operator**: Custom Kubernetes operator managing the agentic system

== Key Technical Insights

=== Prompt Engineering with ADL

Jonathan adopted the **Architecture Definition Language (ADL)** from https://www.developertoarchitect.com/[Mark Richards and Neal Ford] for more structured prompting:

- Moved from informal English prose to semi-formal ADL specifications
- Observed significant improvements in response quality and speed
- Better agent comprehension compared to unstructured prompts

=== Performance Considerations

**Challenges with distributed agents:**
- Initial democratic approach was slow (15+ minutes for simple queries)
- Each agent loading context independently created overhead
- Trade-off between flexibility and performance

**Optimizations:**
- Selective agent participation based on question relevance
- Agents can "bow out" if not relevant to the query
- Running entirely on local GPUs (no cloud API costs)

== Lessons Learned

=== AI-Assisted Development
- Voice-to-text tools (like https://github.com/kitlangton/Hex[Hex]) reduce typing fatigue during long AI sessions
- Maintain discipline: avoid letting AI make imperative cluster changes
- Use Kanban boards to track AI-generated tasks and prevent context loss
- Set clear boundaries in prompts (e.g., "always use GitOps")

=== Distributed Systems
- Cloud-native patterns (messaging, distributed computing) apply to agentic systems
- Kubernetes provides natural orchestration for parallel agent workloads
- Consider hybrid approaches: combine monolithic pipelines with distributed coordination

=== Testing and Metrics
- Measuring prompt quality improvements remains challenging
- Need quantifiable metrics beyond subjective assessment
- Time, token usage, and response quality are key dimensions

== Takeaways for Developers

1. **Quarkus + LangChain4j** provides excellent performance for AI applications with fast startup and low resource usage
2. **Distributed agentic patterns** offer flexibility but require careful design for performance
3. **Local LLM inference** is viable for privacy-sensitive applications with proper hardware
4. **Infrastructure as Code** is essential, especially when working with AI-assisted development
5. **Prompt engineering** benefits from structured approaches like ADL
